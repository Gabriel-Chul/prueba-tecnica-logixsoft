# Prueba Tecnica - Analista Programador

App web simple con:
- Registro e inicio de sesion.
- Mapa con marcadores arrastrables.
- Enfoque en seguridad basica (hash de password, sesiones seguras).

## Stack
- PHP (sin framework)
- HTML/CSS/JS
- Leaflet + OpenStreetMap

## Estructura (inicial)
- app/
  - controllers/
  - services/
- public/
  - assets/css/
  - assets/js/
- storage/

## Notas
- No se usa base de datos; se usara almacenamiento en archivo (JSON).
- Este repositorio seguira Feature Branch Workflow.
